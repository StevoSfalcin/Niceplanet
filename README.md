## Tecnologias

Backend: Node.js

Frontend: Vue.js

# NicePlanet

Projeto desenvolvido com finalidade de teste tecnico.

# Montagem do Ambiente

Backend:

* yarn install
* configurar banco de dados (previante populada com dados fornecidos) no env
* yarn dev

Frontend:

* npm install
* npm run dev
